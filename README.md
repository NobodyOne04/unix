# UNIX LAB

Лабораторные по ОС UNIX

## Lab1

Находится в папке compile, исходник называется script.sh.
Запуск скрипта для сборки происходит следующим образом:

```bash
./script.sh program.c
```

## Lab2

Находится в папке threads, исходник называется main.cpp.
Сборка происходит с использованием cmake файла:

```bash
cd build && cmake ../ && cmake --build .
```

Заранее собраный файл называется run:

```bash
./run
```

## Lab4

Находится в папке automake&autoconf.
Дистрибутив собирается следующей командой:

```bash
./configure && make
```

## Lab5

Находится в папке docker.
Был использован облегченный образ на основе alpine, сборка:

```bash
docker build . -t lab
```

Запуск контейнера с указанием метча портов:
```bash
docker run -p 8080:80 lab
```

Перейти на 0.0.0.0:8080
